### 表结构优化
> 设计表的时候，尽量满足第三范式 <br/>
> 当然也需要为了效率降低要求，增加冗余用空间换时间 <br/>
> 例如游戏商品表里面有四个字段:  <br/>
      -- 游戏道具 --  <br/>
      -- 单个售价 --  <br/>
      -- 销售量 --    <br/>
      -- 总金额 --    <br/>
> 其中总金额可以由单个售价和销售量相乘获得,为冗余字段 <br/>
> 但是这个字段可以提升查询速度,因此可做保留 <br/>
